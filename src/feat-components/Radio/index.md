## Radio 单选框

单选框。

#### 何时使用

- 用于在多个备选项中选中单个状态。
- 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

```tsx
import React from 'react';
import { Radio } from 'bimi-react';

export default () => <Radio title="First Demo" />;
```
